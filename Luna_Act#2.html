Bachelor Science of Information Technology




Programming Fundamentals

This topic focuses on the basic principles and concepts of programming. It covers topics such as variables, data types, control structures (loops and conditionals), functions, and basic algorithms


Database Management Systems (DBMS)

DBMS is a crucial aspect of IT, as it deals with the design, creation, and management of databases. Students learn about database models, normalization, data querying, and database administration.


Network Security

With the increasing dependence on computer networks, network security becomes vital. This topic covers the fundamentals of securing computer networks and systems from unauthorized access, data breaches, and other cyber threats.


Web Development

This topic focuses on the creation and maintenance of websites and web applications. Students learn about web development languages (such as HTML, CSS, and JavaScript), web frameworks, content management systems, and responsive design principles.
